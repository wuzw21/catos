# 内容模板

这个目录是仓库自带的空模板内容目录。

克隆项目后，如果你直接运行：

```bash
npm run serve:demo
```

看到的会是：

- 完整的目录结构
- 空白的内容文件
- 首页里的引导提示

不会包含：

- 作者的真实日志
- 作者的私人关系数据
- 作者的照片和消费记录

你第一次使用时，建议按这个顺序开始：

1. 先补 `soul.md`
2. 再补 `profile/personal-info.md`
3. 再在首页写今天的日计划、打卡和随手记

如果页面里还是空的，就按提示去记录：

- 请先记录你的基本身份
- 请先记录今天真正要推进的一件事
- 请先记录一个有明确日期的重要事项
