# 外部攻略调研

## 说明

2026-04-05 进行了外部调研。由于搜索引擎对小红书原帖的公开索引较差，检索到的高质量、可直接引用内容主要来自 B站公开页面；小红书相关结果更多是活动页、账号页或二次转载页面，因此本轮实操性总结主要基于 B站公开攻略。

## 科研

- 顶会论文和博士科研的内容，反复强调的是：不要把目标只写成“发论文”，而要拆成选题、文献调研、实验、写作、rebuttal 这几个稳定模块。
- 对你更有价值的不是“立刻追三篇”，而是先固定学习时间，把当前已有的一篇主线论文持续推进，再把第二篇、第三篇作为 pipeline 管理。
- 你这一年的关键不是灵感，而是节奏和主线不丢。

## 健身与减脂

- 力量训练内容强调三大项动作规范、渐进超负荷和长期连续性；你当前基线离 1 倍卧推 / 1.5 倍深蹲硬拉还有明显距离，最忌讳的是一边极端减脂一边胡乱冲重量。
- 对你最现实的策略是先把体脂降到 15%-17% 区间，同时维持和抬升绝对力量，再冲专项指标。
- 核心和步频移动不应当被当作“补充”，而要直接并入每周训练安排。

## 篮球 / 足球 / 羽毛球

- 篮球三分训练强调出手节奏、脚步衔接和重复命中，不是单纯站桩投篮。
- 足球长传教学强调支撑脚、触球部位和发力链，先追求稳定 20 米级长传，再提高线路和对抗条件下的准确率。
- 步频、敏捷和多方向移动训练对篮球与足球都有共通价值，可以作为统一体能模块。
- 羽毛球提升若作为挑战项，可以先从核心与步伐打底，不必立刻展开大量专项课时。

## 音乐

- 不跑调训练的高频共识是：先练音准辨别和节奏，再练歌曲。
- 每天 10-15 分钟的短时打卡比偶尔长练更有效。
- 合唱队门槛意味着你不能只会“唱几首歌”，而要有基本音准、节奏、跟伴奏能力。
- 钢琴即兴伴奏作为挑战项，建议晚于音准稳定与基础乐理之后再加速推进。

## 金融

- 记账入门的重点不在复杂模型，而在把账户、收支、预算和储蓄目标分开维护。
- 对你来说，先把 5w -> 10w 存款作为必须项更合理；20w 应该作为挑战项。
- 量化模型可以开始学，但要和真实投资分开：先做研究和回测，再谈资金投入。

## 生活

- 男生护肤相关攻略强调“稳定清洁 + 保湿 + 控油/祛痘”，不建议频繁试产品。
- 穿搭建议集中在干净、简约、统一色系，适合博士生 / 研究者身份。
- 摄影入门内容反复强调先学光线和构图，而不是先买设备。
- 桌搭和居住环境不只是审美，而是为了提高工作和恢复效率。
- 做饭能力最好从高蛋白、低复杂度、能批量复用的食谱开始。

## 本轮可直接采用的结论

1. 科研必须按 pipeline 管，不按灵感管。
2. 健身目标拆成“减脂”和“增力”两个阶段，不硬拧。
3. 步频 / 敏捷 / 核心是运动表现和减伤的共通底层。
4. 音乐先解决音准，再谈合唱和钢琴即兴。
5. 财务先做记账与储蓄系统，再做量化入门。
6. 生活目标里最关键的是建立低摩擦的固定习惯。
