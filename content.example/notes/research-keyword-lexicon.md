# 科研 / 学习 归档词库

## 当前主存位置

- 科研词库主存：`dimensions/03-research/rules.md`
- 学习词库主存：`dimensions/04-learning/rules.md`
- 这份文档现在只做汇总说明，不再作为随手记归档的唯一词库来源。

## 目的

- 这份词库用来帮助系统在归档时区分 `科研` 和 `学习`。
- 目标不是机械匹配，而是先给系统一个稳定的默认判断。

## 学习

这些内容默认更偏 `学习`：

- 上课
- 听课
- 课程
- 作业
- 考试
- 复习
- 教材
- 课程笔记
- 统计学习
- 工具入门

一句话规则：

- 如果目标是掌握课程、完成作业、应对考试，默认归到 `学习`。

## 科研

这些内容默认更偏 `科研`：

- 科研
- 研究
- 论文
- 投稿
- rebuttal
- 实验
- 实验室
- 组会
- 导师
- 汇报
- 开会
- 主线
- 课题
- baseline
- ablation
- method / 方法
- paper / 读 paper
- 学弟
- 学长
- 合作
- 进度对齐
- OmniFlow
- Omni
- 万象
- UTG
- AndroidWorld
- pipeline
- benchmark

一句话规则：

- 如果目标是推动研究产出、实验结果、论文主线、实验室协作，默认归到 `科研`。

## 归档优先级

1. `课程 / 作业 / 考试 / 上课` 优先归 `学习`
2. `论文 / 实验 / 导师 / 组会 / 投稿 / 主线 / 学弟 / 学长 / 实验室` 优先归 `科研`
3. 如果一句话同时含两类内容：
   - 日志里都保留
   - 八维主归类优先选主要目标对应的维度
