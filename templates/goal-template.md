# 目标卡模板

## G-XX 目标名称

| 字段 | 内容 |
| --- | --- |
| 内容 | 这件事要完成什么 |
| 重要性 | 高 / 中 / 低 |
| 投入成本 | 金钱：￥0；时间：每周 0 小时 |
| 发生时间 | 起始时间 / 截止时间 |
| 频率 | 每日 / 每周 / 每月 / 阶段性 |
| 和谁完成 | 独自 / 导师 / 同学 / 朋友 / 家人 |
| 进度条 | `[#####-----] 50%` |
| 当前状态 | 未开始 / 进行中 / 维护中 / 已完成 |
| 最近变化 | 最近一次有实际推进的变化 |
| 下一步行动 | 下一次明确动作 |
